# Pasta de Estética Fixa (Aesthetic) 🌸

Esta pasta é reservada para as decorações permanentes e estéticas do seu site, como o gatinho em cima da barra de pesquisa e as flores de cerejeira nos cantos.

Salve os seus arquivos nessa pasta com os seguintes nomes exatos:

1. **Gatinho em cima da barra:** `popcat.gif` (ou `.png` / `.webp`)
2. **Flores de cerejeira (sakura):** `cherry_blossom.gif` (ou `.png` / `.webp`)

O site está configurado para ler esses dois arquivos diretamente daqui. Caso você mude de ideia ou queira trocar o gatinho/flores por outros gifs decorativos, basta substituir os arquivos nesta pasta usando os mesmos nomes!
