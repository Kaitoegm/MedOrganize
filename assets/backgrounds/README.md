# Pasta de Planos de Fundo (Backgrounds) 🪵

Cole aqui as imagens de fundo para o site. 
Para que o site reconheça seus arquivos automaticamente e os coloque no "Tinder" de descobertas, nomeie os arquivos com os nomes abaixo:

- `bg_1.jpg`, `bg_1.png` ou `bg_1.gif`
- `bg_2.jpg`, `bg_2.png` ou `bg_2.gif`
- `bg_3.jpg`, `bg_3.png` ou `bg_3.gif`
- `bg_4.jpg`, `bg_4.png` ou `bg_4.gif`
- `bg_5.jpg`, `bg_5.png` ou `bg_5.gif`

Você pode usar formatos `.jpg`, `.png`, `.webp` ou `.gif` (imagens estáticas ou animadas). O site é inteligente: se um espaço não tiver imagem, ele simplesmente não exibirá no Tinder para evitar links quebrados!
